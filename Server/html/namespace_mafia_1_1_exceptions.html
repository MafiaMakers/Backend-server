◆ DatabaseWorkingExceptionId

enum Mafia::Exceptions::DatabaseWorkingExceptionId

Возможные id исключений, вызываемых работой с БД

Элементы перечислений
DatabaseWorkingExceptionId_Default 

Базовое значение id исключений работы с БД (сообщений с таким id не бывает)

DatabaseWorkingExceptionId_Backup 

Исключение, вызываемое при ошибке при резервном копировании БД

DatabaseWorkingExceptionId_SQlQuery 

Исключение, вызываемое при ошибке в sql-запросе

DatabaseWorkingExceptionId_EmptyQueryResult 

Исключение, вызываемое при получении пустого ответа на sql-запрос

DatabaseWorkingExceptionId_NotAuthorizedAction 

Исключение, вызываемое при попытке совершить действие, для которого необходима авторизация, без авторизации

DatabaseWorkingExceptionId_TypeMismatch 

Исключение, вызываемое при несовпадении размеров данных, хранящихся в БД и ожидаемых

DatabaseWorkingExceptionId_TimeLimit 

Исключение, вызываемое при превышении времени ожидания отклика БД

DatabaseWorkingExceptionId_UnknownType 

Исключение, вызываемое при передаче неизвестного sql типа данных

DatabaseWorkingExceptionId_Last 

Верхняя граница id исключений работы с БД (см. Mafia::Database::Status_First)

MessageParsingExceptionId_InvalidMessageSize

MessageParsingExceptionId_ControlSumMismatch 

Исключение, вызываемое при получении сообщения с несовпадающей контрольной суммой

MessageParsingExceptionId_NoneKey 

Исключение, вызываемое при попытке дешифрования сообщения дешифратором с незаданным ключом

MessageParsingExceptionId_Last 

Верхняя граница типов исключений парсинга сообщений (см. Mafia::Database::Status_First)

◆ RequestExceptionId

enum Mafia::Exceptions::RequestExceptionId

Возможные id исключений работы системы запросов

Элементы перечислений
RequestExceptionId_Defaut 

Базовый id иключения, связанного с запросами

RequestExceptionId_WrongClient 

Исключение, вызываемое при ответе на запрос не тем клиентом, которому этот запрос предназначался

RequestExceptionId_SizeMismatch 

Исключение, вызываемое при несовпадении размеров типов данных - переданный для получения из ответа запроса и полученный в результате запроса

RequestExceptionId_NotFinished 

Исключение, вызываемое при попытке получить данные из запроса, который еще не завершил свою работу

RequestExceptionId_Last 

Верхняя граница id исключений работы с запросами (см. Mafia::Database::Status_First)
