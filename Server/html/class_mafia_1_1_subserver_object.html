Класс Mafia::SubserverObject

Базовый класс субсервера. Он запускает процесс субсервера, отслеживает краши этого процесса, может отправлять ему запросы и получать сообщения и ответы на запросы. Подробнее...

Подробное описание

Базовый класс субсервера. Он запускает процесс субсервера, отслеживает краши этого процесса, может отправлять ему запросы и получать сообщения и ответы на запросы.

networkerУказатель на объект MainServerNetworker, который отправляет сообщения

maxNotAnsweringМаксимальное количество пропущенных запросов на подтверждение активности субсервера (стандартное значение 5)
specialCommandsСпециальные параметры, которые дополнительно надо передать при запуске субсервера (стандартное значение "")
