◆ Status

enum Mafia::Database::Status

Возможные статусы игрока

Элементы перечислений
Status_First 

Индекс первого статуса. Может использоваться для цикла for:

for(Status status = Status_First; status < Status_Last; status++)

Status_NotConfirmed 

Статус, который появляется у пользователя при регистрации, статус неподтвержденного email-а

Status_Confirmed 

Статус, говорящий о том, что email подтвержден

Status_Last 

Индекс последнего статуса. Может использоваться в цикле for (см. Mafia::Database::Status_First)
