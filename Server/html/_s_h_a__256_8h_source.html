41 USAGE
42 To use these SHA implementations, simply include them in your source with
43 (for example) the following instruction:
44 
45  #include "SHA_lib/SHA256.h"
46 
47 This assumes that SHA256.h is in a directory called SHA_lib, in the same directory
48 as the file you're including SHA256.h in. You can also place SHA256.h in your standard
49 header files/include directory and include it using:
50 
51  #include <SHA256.h>
52 
53 Either way, once you've included the files, the SHA algorithms should be quite
54 easy to use. Simply choose which SHA version you wish to use (the classes are
55 called SHA0, SHA1, SHA256, SHA224, SHA512 and SHA384) and make an instance of
56 the corresponding class, for example:
57 
58  SHA256* yourInstanceName = new SHA256();
59 
60 Then you can use the hash() function, which is in each of the classes,
61 to hash whatever you wish to hash. The hash() function returns a C++ style
62 string (so not a char[]). The hash() function takes either a C++ style string
63 or a char[] as arguments. The char[] version can be used for hashing binary
64 files, C-style strings, or anything else you can convert to a char[].
65 When using the char[] version of hash(), you must also specify the size of the
66 char[] you're passing (or of course the number of elements from it that you
67 want to include in the hash). I recommend that when you calculate the size of
68 a C-style string that you want to hash, you don't include the terminating null
69 character in your calculation, as including it can result in a different hash
70 than expected. Here's how to use the hash() function, short and simple:
71 
72  string digest = yourInstanceName->hash(stringYouWantToHash);
73 
74  or
75 
76  string digest = yourInstanceName->hash(charArrayYouWantToHash, sizeOfCharArray);
77 
78 This will deliver the hash of your input into the string called digest.
79 
80 If you ever want to see which version of the library you're using without going
81 into the source, simply use the shaVersion() method, like this:
82 
83  cout << yourInstanceName->shaVersion() << endl;
84 
85 This will print information about the version of the library onto the screen.
86 The shaVersion() method returns a string, so you can store the version info in
87 a variable as well if you don't immediately want to show it on the screen.
88 
89 Finally, once you're done using SHA, don't forget to delete the instance(s) you
90 made, to avoid memory leaks:
91 
92  delete yourInstanceName;
